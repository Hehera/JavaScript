1.Неявне перетворення та хибні значення в контексті boolean/
Автоматично перетворюється на boolean value,коли воно JS очікував boolean value,а прийшло зовсім інше.

2.Falsy values:
 Значення нижче буде перетворено на false, а все інше буде перетворено на true:
    - false (is already a boolean)
    - null
    - undefined
    - 0
    - NaN
    - "" (empty string) 

 3.У JavaScript як масиви, так і об'єкти є об'єктами. Це означає, що обидва типи даних мають властивості і методи, оскільки вони є екземплярами об'єктів у мові програмування JavaScript. Масиви в JavaScript є особливим типом об'єктів, які використовуються для зберігання послідовних колекцій елементів, в той час як об'єкти використовуються для зберігання пар ключ-значення.
